Titanic Survival Prediction System

Project Overview

This web application predicts the likelihood of survival of a passenger aboard the RMS Titanic using a machine learning model trained on the Titanic passenger dataset.

The system demonstrates the complete workflow of a data-driven web application, from user input and validation to prediction, storage, and result visualization.

The model is trained on historical passenger data and produces probabilistic predictions based on selected passenger attributes.

System Design

The application is structured into multiple pages, each with a specific responsibility:

1. Home Page

Provides an overview of the project, its purpose, and the system architecture.

2. Prediction Form Page

Allows users to enter passenger information such as class, age, sex, and family details.

All inputs are validated to ensure correctness and completeness before submission.

3. Results Page

Displays the predicted survival outcome along with the estimated probability of survival.

The results reflect patterns learned from the training dataset.

4. History Page

Shows previously submitted predictions stored in the system, enabling users to review past inputs and outcomes.

Data and Model

The prediction model is trained using the Titanic passenger dataset, which contains demographic and travel-related attributes.

Missing values are handled during preprocessing, and the model outputs a probability score rather than a binary decision to better represent uncertainty.

Disclaimer

This application is intended for educational purposes only.

Predictions are based on historical data and do not represent certainty or real-world decision-making.
